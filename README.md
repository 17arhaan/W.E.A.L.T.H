# W.E.A.L.T.H: Web-based Expense and Account Ledger for Tracking Habits

## Project Overview
W.E.A.L.T.H is a modern finance tracking application designed to help users manage their personal finances efficiently. It provides an intuitive and secure platform for tracking income, expenses, budgets, and savings goals. The application leverages advanced web development technologies to ensure real-time insights, data security, and a seamless user experience.

## Features
- **Dynamic Dashboard**: Visual representation of financial data using charts and graphs.
- **Transaction Management**: Users can add, edit, and categorize transactions for better financial tracking.
- **Budget Tracking**: Set spending limits and receive alerts when nearing predefined limits.
- **Data Visualization**: Interactive charts for better financial analysis.
- **Secure Authentication**: Implemented encryption and authentication mechanisms for user security.
- **AI-powered Assistant**: Provides real-time insights and assists in navigating the platform.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Privacy-Focused**: Local storage options and encryption ensure user data remains protected.

## Technology Stack
- **Frontend**: React.js
- **Backend**: Express.js (RESTful API)
- **Database**: PostgreSQL
- **Security**: Authentication mechanisms & encryption
- **Visualization**: Chart.js

## Installation and Setup
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/17arhaan/W.E.A.L.T.H.git
   cd wealth-tracker
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Set Up Database:**
   - Install PostgreSQL and create a new database.
   - Configure environment variables for database connection.

4. **Start the Backend Server:**
   ```bash
   npm run server
   ```

5. **Start the Frontend:**
   ```bash
   npm start
   ```

6. **Access the Application:**
   - Open `http://localhost:3000/` in your browser.

## Usage
- Register or log in to your account.
- Add transactions and categorize expenses.
- Set budget limits and monitor spending habits.
- Generate financial reports and insights.

## Security and Privacy
- End-to-end encryption for financial data.
- Secure authentication using industry best practices.
- No third-party data sharing or tracking.
